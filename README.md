# E-commerce Recipe Platform

## Overview
This project is a full-stack e-commerce platform that allows chefs to publish and sell
premium recipes, while users can browse, purchase, and access professional culinary content.

## Tech Stack
- Backend: Spring Boot, Spring Security, JWT
- Frontend: Angular
- Database: MongoDB
- Payments: Stripe
- Email: SendGrid

## Core Features (MVP)
- User authentication (Chef / User)
- Recipe publishing and browsing
- Secure online payments with Stripe
- Access control for purchased recipes
- Purchase history for users
- Revenue dashboard for chefs

## Project Structure
/backend   -> Spring Boot REST API  
/frontend  -> Angular application  
/docs      -> Documentation and diagrams
